Sample Catalogue:
 
The following information is provided for each sample:
        PI                    Initials of the investigator who provided the sample.
        Relab Sample ID       Unique identification of each sample.  The format is
                                   XY-PII-NNN-ABC where
                                   XY = Subcode
                                   PII = PI initials
                                   NNN = A sequence number
                                   ABC = Optional characters or numbers
        Sample name           The name given the sample by the PI; lunar sample number, etc.
        PO                    Place of origin:  TE=terrestrial; LU=lunar; XT=extraterrestrial
        General type          Broad classification scheme (soil, rock, experimental, etc.)
        Type                  intermediate classification
        Subtype               most specific classification (mineral, rock type, etc.)
        Min Size              minimum particle size; zero if rock, slab, or thinsection
        Max Size              maximum particle size; zero if rock, slab, or thinsection
        Texture               the physical form of the sample (chip, particulate,thinsection, etc.)
        Origin                Site or location from where sample originated
        Text                  comments
        Chem #                Number of the chemical analysis file, when available
 
 
Spectra Catalogue:
 
The following information is provided for each spectrum:
        PI                    Initials of the investigator who provided the sample.  
        Subcode               Subcode for sample ID
        Relab File            file name containing the spectrum [needed to find spectrum in Data folders]
        Relab Sample ID       Unique identification of each sample.
        Date                  Date spectrum was collected
        Start Wave            Starting wavelength for the spectrum in nanometers
        Stop Wave             Ending wavelength for the spectrum in nanometers
        Resl                  Sampling resolution in nanometers
        Source                Angle of illumination in degrees
        Detect                Angle of emergence or detection in degrees
 
 
Investigator Addresses
       The name and mail addresses and email addresses are provided for RELAB 
investigators (principal and secondary).  
 
 
Chemical Analyses
        Major and minor elemental abundances are available for some samples.  The 
source of the information is provided if the data were obtained from the literature.
